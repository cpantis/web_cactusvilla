# Cactus Villa Pro

Clean Next.js project prepared for a new GitHub repo and Vercel deploy.

## Run locally

```bash
npm install
npm run build
npm run dev
```

## Deploy

1. Create a new GitHub repository.
2. Upload all files from this folder.
3. Import the repo into Vercel.
4. Deploy.

## Notes

- Next.js is pinned to **15.5.7** to avoid the blocked vulnerable versions on Vercel.
- `node_modules` and `.next` are ignored in `.gitignore`.
- Replace `/public/anette.png` with a more polished portrait later if desired.
- Replace the Gallery page placeholders with actual property photography when available.
